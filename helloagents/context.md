# 项目上下文

## 项目定位
`mini-im` 是一个轻量 IM 项目，后端提供 HTTP API 与 WebSocket（WS）实时通道，前端提供 Web 端 UI（仿微信绿白风格）。

## 目录结构（核心）
- 后端：`src/main/java/`、`src/main/resources/`
- 前端：`frontend/`（Vue 3 + TypeScript + Vite）
- 知识库：`helloagents/`（CHANGELOG / wiki / history）

## 本地开发（常用）
- 后端：`mvn spring-boot:run`
- 前端：`npm -C frontend run dev`
